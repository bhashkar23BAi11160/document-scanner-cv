# 📄 Document Scanner using OpenCV

## 🔍 Problem Statement
Capturing documents using a mobile camera often results in skewed, noisy, or poorly lit images. This reduces readability and usability.

## 💡 Solution
This project detects document boundaries using contour detection and applies perspective transformation to generate a clean, scanned version of the document.

## ⚙️ Technologies Used
- Python
- OpenCV
- NumPy

## 🚀 Features
- Edge detection (Canny)
- Contour detection
- Perspective transformation
- Noise reduction
- Binary thresholding for clean output
