# 📄 Document Scanner using OpenCV

## 🔍 Problem
Mobile photos of documents are often skewed, noisy, and unclear.

## 💡 Solution
This project detects document edges and applies perspective transformation to generate a clean scanned output.

## ⚙️ Tech Used
